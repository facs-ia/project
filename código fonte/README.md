# Código Fonte

Nesta pasta encontram-se todos os notebooks utilizados nos treinamentos feitos utilizando o **MLPClassifier**.

O notebook com os resultados apresentados no poster é: [ia_a3_mlp_notebook_13_11_2025 - treino 1.ipynb](ia_a3_mlp_notebook_13_11_2025_treino_1.ipynb)

Todos os arquivos podem ser executados no Google Colab. O Github também é capaz de exibir os últimos resultados processados em cada notebook.

Todos os notebooks possuem as bibliotecas necessárias para serem executadas no Google Colab. Os notebooks também possuem um código para baixar o arquivo .CSV do dataset automaticamente, hospedado em outro ambiente.

Caso ocorra algum erro no download automático, o dataset pode ser encontrado também neste repositório: [dataset-credit_card_fraud_data.zip](dataset-credit_card_fraud_data.zip)

ou na sua fonte original: https://www.kaggle.com/datasets/dhanushnarayananr/credit-card-fraud

Caso opte por baixar manualmente o dataset, o arquivo .zip deve ser descompactado. E caso esteja executando o notebook na nuvem (ex Google Colab), o arquivo .CSV descompactado deve ser enviado para o ambiente na nuvem.
